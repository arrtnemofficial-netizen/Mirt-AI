# Contributing (UA)

## Гілки

- Фічі: `feat/*`
- Фікси: `fix/*`

## Форматування та лінтінг

```bash
ruff format .
ruff check .
```

## Тести

```bash
pytest
```

## Змінні середовища

Файл `.env` не додається до git. Використовуйте `.env.example` як шаблон.

## Pull Request

- Переконайтеся, що всі тести проходять.
- Додайте опис змін до PR.
- Дотримуйтесь існуючого стилю коду.
