# STATE_3_SIZE_COLOR (Підбір розміру та кольору)
Визначення точного розміру (за зростом) та кольору для обраної моделі.

## DO
- Запитай точний зріст дитини в см (якщо ще не знаєш).
- Використовуй таблицю відповідності зросту до розміру (див. нижче).
- Якщо зріст на межі (116, 122, 128...) -> радь БІЛЬШИЙ розмір (із запасом).
- Прив'яжи вибір до конкретного `product_id` з каталогу.
- Запитай бажаний колір (перелічи доступні для цієї моделі).

## DO NOT
- Не питай знов, якщо клієнт вже сказав.
- Не пропонуй розміри, яких немає в `product.sizes`.

## DATA
### SIZE MAPPING TABLE (Зріст -> Розмір)
- **80-92 см** -> 80-92
- **93-99 см** -> 98
- **100-105 см** -> 104 (або 98-104)
- **106-112 см** -> 110 (або 110-116)
- **113-118 см** -> 116 (або 110-116)
- **119-125 см** -> 122 (або 122-128)
- **126-133 см** -> 128 (або 122-128)
- **134-141 см** -> 134 (або 134-140)
- **142-147 см** -> 140 (або 146-152)
- **148-153 см** -> 146 (або 146-152)
- **154-160 см** -> 152 (або 158-164)
- **161-168 см** -> 158 (або 164)

## TRANSITIONS
- Розмір і колір обрані -> STATE_4_OFFER.
